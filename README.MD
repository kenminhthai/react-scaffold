A lightweight, opinionated setup to get up and going on react projects using
* react
* redux
* react-router
* webpack
* babel
* aphrodite


###Getting Started
* $npm install
* $npm start

###Debugger
install this [chrome extension](https://github.com/zalmoxisus/redux-devtools-extension)
